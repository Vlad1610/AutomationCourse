Environment variables

  • System variables
  • User variables
  • Path variable

JAVA SDK

Go to Oracle.com
  1. Downloads -> Java SE -> version 8xxx
  2. Install
  3. Add the JDK path to system PATH environment variable
  4. Test the installation with command

Java - version

Apache Maven

Go to Apache Maven Website
  1. Download the latest binary zip
  2. Unzip to a directory of you choice
  3. Add the JDKAdd the bin dir of the installation to PATH
  4. Create and new env variable called JAVA_HOME pointing to the JDK path
  5. Add the JDKTest the Maven installation with command

mvn --version

IntelliJ IDEA

Go to JetBrains.com
  1. Download IntelliJ IDEA Community Edition
  2. Install

GIT

Go to Git.com
  1. Create a new user environment variable called HOME, pointing toyour user’s home directory (C:\Users\username)
  2. Add the GIT install path to PATH environment variable
  3. Test the GIT installation with command git
  4. Configure GIT with commands

git config --global user.name “your user”

git config --global user.email “your email”
